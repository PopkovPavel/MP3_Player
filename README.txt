[RU]
Для установки плеера нужно открыть установочный файл (MP3_Player.exe) и указать путь установки: 
"C:\" (Без кавычек)

Плеер запускает любые mp3 файлы, хранящиеся у вас на компьютере.

Доступно управление через клавиатуру:
[down] Громкость тише на 5%
[up] Громкость выше на 5%
[left] Перемотать назад на 5 сек
[right] Перемотать вперёд на 5 сек
[space] Пауза/продолжить
[a] Предыдущий трек
[d] Следующий трек
[r] Повтор (переключение режима)
[q] Случайный трек
[f] Выбор папки
[esc] Выход из приложения

ПРИМЕЧАНИЕ
Установленное приложение работает ТОЛЬКО из директории "C:\" 
и ТОЛЬКО с наличием папки "images" со всеми её вложениями!

Также возможны случаи, когда Защитник Windows посчитает этот файл вредноносным или даже опасным (из-за отсутствия цифровой подписи)
(в таком случае нужно добавить .exe файл в список исключений антивируса)

ВНИМАНИЕ
В случае некорректоной работы приложения необходимо файл "libmpg123-0.dll" поместить в каталоги system32 и sysWOW64 
(C:\Windows\System32 и C:\Windows\SysWOW64)

Также для удобства можно создать ярлык приложения и поместить его в удобное для себя место.
